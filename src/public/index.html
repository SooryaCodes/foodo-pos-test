Food Ordering POS Backend

RESTful API for Inventory Management

Overview

A backend service built with pure Node.js and TypeScript for managing inventory in a Food Ordering POS system. This project demonstrates core backend fundamentals including HTTP handling, data modeling, CRUD operations, and comprehensive audit logging.

Core Features

  • Item and Variant management with flexible properties
  • Complete audit trail for all data changes
  • Soft delete functionality for variants
  • Filterable audit logs by item or variant
  • Interactive Swagger API documentation

Tech Stack

Node.js TypeScript MongoDB Atlas Swagger Docker
